BALL SURFERS (Unknown)

BALL SURFERS is an cross-platform multiplayer and singleplayer game in which you have to avoid obsticle by moving left or right and you can also compare your highscore with other players. In multiplayer you have an unlimited obsticle map but with an ending which is impossible to complete

System Requirements

Android

  • RAM = 1GB+ RAM
  • Disk Space = 80mb
  • CPU = ARMv7 with Neon Support (32-bit) or ARM64
  • Graphics API = OpenGL ES 2.0+, OpenGL ES 3.0+, Vulkan
  • Operating system = Android 4.4 (API 19)+

Windows

  • RAM = 2GB+ RAM
  • Disk Space = 180mb
  • CPU = x86, x64 architecture with SSE2 instruction set support
  • Graphics API = DX10, DX11, DX12 capable
  • Operating system = Windows 7 (SP1+) , Windows 10 and Windows 11
